# Отчёт: мониторинг HQ-SRV и BR-SRV

## Выбранное программное обеспечение

Использованы Prometheus, Grafana и Node Exporter. Все компоненты являются
открытым программным обеспечением.

Prometheus собирает и хранит метрики, Node Exporter предоставляет системные
метрики Linux, а Grafana отображает их на готовом dashboard.

## Основные параметры

| Компонент | Порт | Назначение |
|---|---:|---|
| Grafana | 3000/tcp | Веб-интерфейс мониторинга |
| Prometheus | 9090/tcp | Сбор и хранение метрик, доступ только с HQ-SRV |
| Node Exporter | 9100/tcp | Метрики HQ-SRV и BR-SRV |

Адрес Grafana: `http://mon.au-team.irpo:3000`.

Учётная запись: `admin`, пароль: `P@ssw0rd`.

В Prometheus настроены цели:

- HQ-SRV: `192.168.1.10:9100`;
- BR-SRV: `192.168.3.10:9100`.

Dashboard визуально отображает загрузку CPU, занятую оперативную память и
занятое место основного накопителя.

Для разрешения имени в `/etc/dnsmasq.conf` HQ-SRV добавлена запись:

```text
address=/mon.au-team.irpo/192.168.1.10
```
