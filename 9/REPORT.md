# Отчёт: защита SSH с помощью fail2ban

На HQ-SRV настроен fail2ban для службы SSH.

Основные параметры:

| Параметр | Значение |
|---|---:|
| SSH-порт | 2026/tcp |
| Количество ошибок до блокировки | 3 |
| Интервал подсчёта ошибок | 600 секунд |
| Время блокировки | 60 секунд |
| Backend | systemd |

Конфигурация расположена в `/etc/fail2ban/jail.d/sshd.local`.

Состояние защиты проверяется командой:

```bash
fail2ban-client status sshd
```
