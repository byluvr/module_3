С GitHub

Выполнить:

 apt-get update && apt-get install git -y
 git clone https://github.com/byluvr/module_2
 git clone https://github.com/byluvr/module_3

На каждой машине!

В папке module_2 и module_3 есть подкаталоги с заданиями, в каждом каталоге задания лежат скрипты, они пронумированы и в названии указано где их запускать, так же читайте README.md в каждом каталоге, тк иногда требуются ручные действия!
Через wget

 apt-get update && apt-get install wget -y
 wget -r -np -nH --cut-dirs=1 --user=files --password=1E2m3i4l5H6a7z! http://wiki-altlinux.ru/files/

Скачаются не только скрипты, а еще лишние файлы по типу index и robots, их по окончанию шалости и запуска скрипта очистки, тоже нужно удалить вручную:

 rm -rf названия файлов

По ssh

 scp -r root@5.35.82.215:/srv/downloads/ .

Внимание! Точка в конце обязательна!

Пароль: 1E2m3i4l5H6a7z!

После завершения пакостей, и после запуска скриптов очистки (clear), вручную удаляем лишние файлы или каталоги:

  rm -rf downloads

В каждом каталоге каждого задания есть файл README.md, чаще всего там написано что то полезное, обязательно читать!
так же, если вдруг в скрипте нужно что то поменять, вы сами файлы .sh ни в коем случае не лезьте, всё настраивается через виртуальное окружение .env

--- vim .env 

читайте README и снапшоты обязательно так же делайте
